# Comick Tierlist Generator

> **⚠️ ARCHIVED PROJECT ⚠️**  
> This project is no longer functional as **Comick.io has been shut down**. This repository is kept for archival purposes and code reference only.

A Next.js application that uses the Comick.io API to pull a user's comics onto the website to create a tierlist.

## Getting Started

### Prerequisites

- Node.js 18.0 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/GooglyBlox/comick-tierlist-gen.git
cd comick-tierlist-gen
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
